# Kennel design
The design information is shown below.

## Components
* `main.go`
    * Initialization
    * Main loop
* `user`
    * User interface (e.g InsertChr, Linefeed)
* `window`
    * window control
* `buffer`
    * Buffer control
* `file`
    * File management
* `debug`
    * Debug interfaces

## Tips
* tips
    * The current terminal window size can be obtained with `termbox.Size()`
